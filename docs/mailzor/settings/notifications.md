# Notification Templates

Notification templates allow you to customize the automated emails sent to your customers and staff when specific events occur within the platform.

## Key Features

- **Dynamic Variables**: Use placeholders like `{{ $name }}` or `{{ $order_id }}` to personalize content for every recipient.
- **Rich-Text Editor**: Compose professional emails with a full-featured visual editor.
- **Versioning**: Duplicate templates to test different versions or messaging.

## Managing Notifications

### Edit a Notification

1. From your admin dashboard, go to **Settings > Notifications**.
2. Click the **name** of the template you wish to modify.
3. Update the **Subject**, **Content**, and other settings.
4. Click **Save**.

### Duplicate a Template

To create a new template based on an existing one:

1. Click the action menu (**...**) next to the notification.
2. Select **Duplicate**.
3. Enter a new title and choose which details to copy.
4. Click **Save**.

### Setting a Default Template

If you have multiple versions of a notification (e.g., two different Welcome emails), you must specify which one is currently active.

1. Click the action menu (**...**) next to the desired template.
2. Click **Mark as Default**.

### Delete a Template

::: danger
Deleted notification templates cannot be restored.
:::

1. Click the action menu (**...**) next to the template.
2. Select **Delete**.
3. Confirm the action.

::: warning NOTE
Templates currently marked as **Default** cannot be deleted. You must mark another template as default first.
:::
