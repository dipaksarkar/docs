# Plans & Pricing

Manage your SaaS subscription tiers, feature limits, and recurring billing cycles.

## Configuring Subscription Plans

Each plan defines what your customers can access and how much they are charged.

- **Label**: The public-facing name of the plan (e.g., `Pro`, `Enterprise`).
- **Price**: The recurring cost of the subscription.
- **Setup Fee**: An optional one-time charge applied when the user first subscribes.
- **Interval**: Choose between `Monthly`, `Quarterly`, `Semi-Annually`, or `Yearly` billing.
- **Contract**: If enabled, users are committed to a minimum term.
- **Freeze Fee**: The cost to temporarily pause a subscription instead of canceling it.
- **Grace Period**: Number of days a user remains active after a failed payment before access is revoked.

## Feature & Usage Limits

Control the scale of access for each plan tier:
- **Max Contacts**: The maximum number of subscribers a user can have.
- **Campaign Limit**: Monthly cap on the number of email campaigns.
- **Automation Access**: Toggle whether advanced visual workflows are available.

## Managing Plans

### Add a Plan
1. Navigate to **Settings > Plans & Pricing**.
2. Click **Add Plan**.
3. Define the pricing, interval, and limits.
4. Set the status to **Active** to make it available for new subscribers.

### Edit a Plan
Modifying a plan's price will only affect new subscribers. Existing subscribers will continue on their original terms unless manually updated.

### Deleting Plans
::: warning
Plans can be soft-deleted. Deleting a plan prevents new sign-ups but does not automatically cancel existing subscriptions associated with that plan.
:::
