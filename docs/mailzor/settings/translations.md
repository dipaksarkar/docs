# Translations & Localization

Make Mailzor accessible to a global audience by managing application translations directly from the dashboard.

## Managing Languages

- **Primary Language**: The default language for new users, set in [General Settings](./general.md).
- **Supported Languages**: A list of all active languages available for the interface.

## Translating the Interface

You can update every string in the application to match your brand's voice or a specific dialect.

1. Navigate to **Settings > Translations**.
2. Select the language you wish to edit from the sidebar.
3. Use the **Search** bar to find specific phrases or buttons (e.g., "Login", "Campaign").
4. Enter the new translation in the text field.
5. Click **Save** to apply the changes instantly across the platform.

## Adding a New Language

To add a language that isn't currently in the system:
1. Click **Add Language**.
2. Select the language and locale (e.g., `Spanish - es`).
3. The system will create a copy of the default (English) strings, which you can then translate manually.

## Exporting & Importing
(Optional: If supported by the UI)
You can export translation files to work on them locally and import them back once complete.
