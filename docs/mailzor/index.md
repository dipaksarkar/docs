## Core Features

### Hybrid Email Delivery

- **Amazon SES Integration**: Primary delivery path for maximum deliverability and scale.
- **Custom SMTP Fallback**: Automatic failover to secondary SMTP servers if SES limits are reached.
- **Real-time Webhooks**: Automated bounce and complaint handling via AWS SNS.

### Campaign & Automation

- **Multi-step Campaign Builder**: Intuitive workflow for creating, scheduling, and sending campaigns.
- **Visual Email Designer**: Drag-and-drop builder with pre-designed sections and blocks.
- **Automated Workflows**: Linear event-driven automations (e.g., welcome sequences, follow-ups).

### List Management & Validation

- **Bulk Contact Management**: Easy import/export with custom tagging and segmentation.
- **Email Validation Engine**: Built-in single and bulk validation with catch-all and disposable detection.
- **Credit-Based System**: Flexible monetization for email validation and campaign sends.

### Analytics & Tracking

- **Real-time Metrics**: Open rates, click rates, delivery rates, and bounce tracking.
- **Activity Timelines**: Hourly engagement data for campaigns.
- **Link-Level Tracking**: Detailed performance metrics for every URL in your emails.

### Admin Tools

- **Complete SaaS Management**: Manage plans, users, staff, and system configurations.
- **Queue Monitoring**: Full visibility into background job processing via Laravel Horizon.
- **Comprehensive Logs**: Detailed audit trails for all system and user activities.
